# Статус проекта AlexWoW

> **Дата среза:** 2026-06-13 · **Возраст проекта:** ~12 дней (старт 02.06.2026) ·
> **Темп:** 248 коммитов ≈ **~20 коммитов/день**.
>
> **Сводная готовность (взвешенная по доменам): ≈ 58%.**
> Игровое ядро (войти в мир, бой, спеллы, квесты, лут, профессии, прогрессия) — готово
> и проверено клиентом. Остаток — социальные/эндгейм-системы (гильдии, группы, PvP,
> петы, аукцион, почта).

Зачем этот файл: единая сводка «где мы». Плоский счётчик опкодов (171/863 = 19.8%)
занижает картину — опкоды неравновесны, «заведён» ≠ «работает», а крупные пласты без
опкодов (навмеш, статы, ИИ, БД, веб) в 863 не попадают. Поэтому **основная метрика —
домены геймплея**, опкоды — деталь внутри домена, вехи — историческая канва.

Этот файл — **основной** для статуса проекта. Историческая детализация (вехи M0–M11,
полные таблицы опкодов, дневники) перенесена в архив и обновляется по необходимости:
[archive/roadmap.md](archive/roadmap.md) · [archive/opcodes-Implemented.md](archive/opcodes-Implemented.md) ·
[archive/opcodes-NotImplemented.md](archive/opcodes-NotImplemented.md) · [archive/devlog/](archive/devlog/).

## Скорость и прогноз

- **Темп.** ~20 коммитов/день, 12 вех (с историей M0→M11) закрыты за ~2 недели — высокая
  скорость на ядре, т.к. есть проверенный эталон (CMaNGOS `C:\repo\mangos-wotlk`).
- **Что покрыто.** Весь «играбельный минимум» + глубина персонажа (статы, таланты,
  спелл-механики, профессии). Это ~60% взвешенной работы и самые трудоёмкие XL-домены.
- **Что осталось (по убыванию веса).** Гильдии, Петы, PvP (BG/арена) — XL; затем
  Социалка, Группы/рейды, Почта, Аукцион — L. Это **широта контента**, а не ядро:
  каждая система относительно изолирована и опирается на уже готовый каркас
  (опкод-диспетчер, БД, реестр мира).
- **Текущий фокус** (решено 2026-06-09): глубина персонажа → завершение профессий, далее
  классовые механики (руны DK, крит от статов, ICD проков). См. roadmap «Дальше».
- **Грубая оценка остатка.** 15 непокрытых доменов ≈ 39 из 111 весовых баллов (~35%).
  При сохранении темпа и наличии эталона широта набирается быстро, но XL-системы
  (гильдии/петы/PvP) — самые ёмкие; реалистично это основной объём оставшейся работы.

## Как обновлять этот файл

1. При добавлении опкодов в `src/AlexWoW.WorldServer/Protocol/WorldOpcode.cs` —
   синхронизировать [archive/opcodes-Implemented.md](archive/opcodes-Implemented.md) и
   [archive/opcodes-NotImplemented.md](archive/opcodes-NotImplemented.md) (счётчики + категории).
2. При закрытии среза работы — обновить соответствующий домен в таблицах выше
   (историческая канва вех — в [archive/roadmap.md](archive/roadmap.md)).
3. Статус домена ставится **по факту** (работает у клиента), а не по проценту
   опкодов. Прогресс по доменам ведётся на канбан-доске `https://alexwow.home.srv/Board`
   (проекты `P[N] — <домен>`). Пересчитать сводный
   взвешенный % (вес: S=1, M=2, L=3, XL=5; кредит: ✅=1.0, 🟡=0.5, ⬜=0).
