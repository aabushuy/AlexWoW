# Документация AlexWoW

Самописный сервер World of Warcraft на **.NET 10** + **MySQL**.
Цель — играбельный сервер под **WotLK 3.3.5a (build 12340)**. Эталон — CMaNGOS-WotLK.

## Как ведём документацию

Прогресс ведётся **по доменам геймплея** (см. [status.md](status.md)) — это основная
ось. Активная работа по доменам — на канбан-доске `https://tasks.home.srv` (проекты
`P[N] — <домен>`, где N — порядок выполнения).

- **`status.md`** — спина проекта: дашборд доменов со статусами и сводным %.
- **`architecture.md`** — относительно стабильный справочник по дизайну; меняется реже.
- **`archive/`** — историческая канва: вехи M0–M11 (`roadmap.md`), полные таблицы
  опкодов, дневники разработки по вехам (`devlog/<веха>.md`). Пополняется по необходимости.

Правило: закрыли срез домена → обновили домен в `status.md` и задачу на канбане.
