# Документация AlexWoW

Самописный сервер World of Warcraft на **.NET 10** + **MySQL**.
Цель — играбельный сервер под **WotLK 3.3.5a (build 12340)**. Эталон — CMaNGOS-WotLK.

| Документ | О чём |

| [Дорожная карта](roadmap.md) | Вехи M0–M10 + веб-панель M8, статусы, критерии приёмки |

## Как ведём документацию

Документация организована **по вехам (milestones), а не по спринтам**.

- **Веха** = законченный кусок, дающий *проверяемое поведение клиента* (M1 → экран
  реалмов, M2 → вход в мир). Объём вех разный, поэтому фиксированные спринты не подходят.
- **`roadmap.md`** — спина проекта: список вех со статусами и критериями приёмки.
- **`devlog/<веха>.md`** — бортовой журнал вехи: что реализовано, какие приняты
  решения и на что напоролись (роль спринт-ретро, но привязка к вехе, а не к календарю).
- **`architecture.md`** — относительно стабильный справочник по дизайну; меняется реже.

Правило: завершили веху → обновили её статус в `roadmap.md` и дописали `devlog/`.
