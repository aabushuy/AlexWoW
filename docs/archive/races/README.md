# Расы — трекинг расовых абилок (WoW 3.3.5a)

Файл на расу: расовые абилки (активные + пассивные). Что проверено клиентом, что осталось.
Легенда статуса — как в [classes/README.md](../classes/README.md): ✅ проверено · 🟡 реализовано · ⬜ не сделано · ➖ вне этапа.

Колонки: **Абилка** · **Спелл** (id) · **Школа/Аура/Эффект** · **Тип** (Active/Passive) · **Статус**.

Расовые — обычно особые механики (иммунитеты, snare-removal, AoE-stun, restore ресурса), поэтому почти всё ⬜:
их надо распознавать в `SpellCatalog`/эффектах отдельно. Фракции: Альянс = Человек/Дворф/Ночной эльф/Гном/Дреней;
Орда = Орк/Нежить/Таурен/Тролль/Эльф крови.
