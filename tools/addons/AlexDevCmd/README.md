# AlexDevCmd — клиентский пульт dev-команд (WoW 3.3.5a)

Древовидная панель для расстановки dev-сущностей, прокачки и **телепорта по городам** без ручного ввода
в чат. **Каталог меню отдаёт сервер** — аддон ничего не хардкодит (#79).

## Как работает
- **Каталог с сервера.** При входе в мир аддон шлёт addon-сообщение `AlexDev menu`
  (`SendAddonMessage("AlexDev","menu","WHISPER",<имя>)`). Сервер (admin-гейт `is_admin`) отвечает
  серией addon-сообщений `AlexDev\t<строка-узла>`, кадрированных `BEGIN`/`END`. Аддон строит дерево из
  узлов `N|id|parent|kind|label|payload…` и рисует его. Транспорт — addon-сообщения поверх чат-опкодов
  3.3.5a (язык `LANG_ADDON`), в чат не печатается.
- **Команды-листья** уходят как обычный **SAY**-чат (`.trainer mage`, `.level 70`…). Сервер ловит `.`
  (`DevCommands.TryHandleAsync`) и не эхоит. Гейт `is_admin` — на сервере, аддон ничего не обходит.
- **Листья-ввод** («Уровень…», «Выдать предмет…», «Изучить спелл…», «Баффы») открывают окно ввода;
  второй аргумент — через пробел (`1234 5`, `6673 300`).
- **Телепорт.** Ветка «Телепорт» (Альянс/Орда/Нейтральные) подтягивается из БД (`alexwow_auth.dev_teleport`).
  Клик по городу → поп-ап «Телепортироваться в CITY? Да/Нет» → `.tp <id>`. Та же карта — мгновенно, другая
  — через загрузочный экран.

## Открыть/закрыть
- **Кнопка у миникарты** (иконка телепорта) — клик открывает/закрывает пульт; перетаскивание — вокруг
  миникарты (позиция сохраняется).
- Слэш **`/dev`** (или `/devcmd`).
- Категории сворачиваются/разворачиваются кликом (`+`/`-`), глубина произвольная.

Позиция окна и угол кнопки сохраняются (SavedVariable `AlexDevCmdDB`).

## Менять города телепорта
Состав/координаты/порядок — в таблице `alexwow_auth.dev_teleport` (`sort_order`, `name`, `faction`
0/1/2, `map`, `x`, `y`, `z`, `o`). Аддон тянет список при входе — правки в БД видны после `/reload` или
повторного входа.

## Установка
Скопировать папку `AlexDevCmd` в `Interface\AddOns\` клиента (напр. `D:\Games\WoW335\Interface\AddOns\`)
и сделать `/reload`. Включить аддон в списке на экране персонажей, если выключен.

Исходник ведётся в репозитории (`tools/addons/AlexDevCmd/`); клиентская копия — отдельно.
