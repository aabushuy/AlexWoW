# AlexDevCmd — клиентский пульт dev-команд (WoW 3.3.5a)

Древовидная панель для расстановки dev-сущностей и прочих dev-команд AlexWoW без ручного ввода в чат.

## Как работает
Аддон отправляет команды как обычный **SAY**-чат (`.trainer mage`, `.level 70` и т.п.). Сервер ловит
сообщения, начинающиеся с `.` (`DevCommands.TryHandleAsync`), и не эхоит их в чат. **Гейт `is_admin`
остаётся на сервере** — аддон ничего не обходит, под обычным аккаунтом команды отклонит сервер.

- Открыть/закрыть панель: **`/dev`** (или `/devcmd`).
- Категории сворачиваются/разворачиваются кликом (`+`/`-`).
- Лист с готовой командой — клик сразу выполняет.
- Лист с «…» (Уровень…, Выдать предмет…, Изучить спелл…, Баффы) — открывает окно ввода; для
  `.additem`/`.buff` второй аргумент вводится через пробел (`1234 5`, `6673 300`).
- Позиция окна сохраняется (SavedVariable `AlexDevCmdDB`).

Каталог команд захардкожен в `AlexDevCmd.lua` (таблица `TREE`) — при добавлении новых dev-команд на
сервере дополнить её.

## Установка
Скопировать папку `AlexDevCmd` в `Interface\AddOns\` клиента (напр. `D:\Games\WoW335\Interface\AddOns\`)
и сделать `/reload` (или перезапустить клиент). Включить аддон в списке на экране персонажей, если выключен.

Исходник ведётся в репозитории (`tools/addons/AlexDevCmd/`); клиентская копия — отдельно.
